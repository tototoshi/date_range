# date_range

## Install

```
$ curl https://raw.githubusercontent.com/tototoshi/date_range/master/date_range > ~/bin/date_range && chmod 0755 ~/bin/date_range
```


## Usage

```
$ date_range --start 20130301 --end 20130401 --step 3
20130301
20130304
20130307
20130310
20130313
20130316
20130319
20130322
20130325
20130328
20130331
```
